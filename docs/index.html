How it works

PixelPick is a small always-on-top tool. Move your mouse over any pixel on screen and the color updates in real time. Freeze the display when you find the right color, then copy it in the format you need.

1

Hover

Move your mouse over any pixel. PixelPick shows the color and a ×10 zoom of the area around the cursor.

2

Freeze

Press Ctrl + Space to freeze the display. Use arrow keys to nudge the cursor one pixel at a time for precision.

3

Copy

Click the copy button next to HEX, RGB, or HSL to put the value on the clipboard.

Color formats

PixelPick displays the sampled color in three formats, each with a one-click copy button.

HEX

#A855F7

Standard web hex format, uppercase. Ready to paste into CSS, Figma, or any design tool.

RGB

168, 85, 247

Red, green, and blue channels as decimal values 0–255.

HSL

270°, 91%, 65%

Hue, saturation, and lightness — useful for reasoning about color relationships.

Keyboard shortcuts

ShortcutAction
Ctrl + Space Freeze / unfreeze — lock the display on the current color
↑ ↓ ← → Nudge the cursor one pixel at a time (while not frozen)
Ctrl + ↑ / ↓ Increase / decrease zoom level (×5 – ×15, default ×10)

Arrow keys are intercepted only while PixelPick is active and not frozen, so they do not interfere with other applications. When frozen, arrow keys pass through normally.

Editing colors

When frozen, the Edit button opens the Windows color picker so you can adjust the sampled color. The sample panel splits to show the original color on the left and your edited version on the right.

Click Revert at any time to go back to the original sampled color.

Privacy Policy

PixelPick does not collect, store, or transmit any personal data.

  • ✓ No screen recording: Screen pixels are sampled in real time at the cursor position only and never stored or transmitted.
  • ✓ No network access: PixelPick makes no network connections of any kind.
  • ✓ No telemetry: No usage data, crash reports, or analytics are collected.
  • ✓ No third-party services: PixelPick has no dependencies on external services or APIs.
